CSS-布局之flex布局

#参考资料：MDN文档，阮一峰的文章
阮一峰：Flex 布局教程：语法篇

随心追梦：Flexbox完整指南- A Complete Guide to Flexbox

巩固flex知识的小游戏1

巩固flex知识的小游戏2

#1、预备知识：

  • flex容器中默认存在两条轴，主轴（main axis）和交叉轴（cross axis）；
  • flex容器是flex container，每个项目称为flex item；
  • 主轴是水平方向时，每个flex item占据的主轴空间为main size，占据的交叉空间为cross size；
  • 主轴起点是main start，终点是main end；
  • 交叉轴起点是cross start，终点cross end ；
  • 制作flex盒子：.container{display: flex/inline-flex}
    块状元素用flex，行内元素 用inline-flex；
  • 设成了flex盒子后，子元素的float、clear、vertical-align属性都会失效！

#2、flex容器（flex container）的属性

  1. flex-direction —-“主轴的方向”

    属性：

    • row—-“水平方向，起点在左” 默认值
    • row-reverse —-“水平方向，起点在右”
    • column—-“垂直方向，起点在上”
    • column-reverse —-“垂直方向，起点在下”

  2. flex-wrap —-“是否换行”

    属性：

    • nowrap—-“不换行” 默认值
    • wrap—-“换行”
    • wrap-reverse —-“换行，首行在下”

  3. justify-content —-“容器里项目 主轴对齐方式”

    属性：

    • flex-start —-“靠向起点” 默认值
    • flex-end —-“靠向终点”
    • center —-“居中”
    • space-between —-“两端对齐”
    • space-around —-“等间隔对齐”

  4. align-items —-“容器里项目 交叉轴对齐方式”

    属性：

    • strentch —-“延伸” 默认值
    • flex-start —-“靠向起点”
    • flex-end —-“靠向终点”
    • center —-“居中”
    • baseline —-“项目第一行文字的基线对齐” 不常用

  5. align-content —-“项目在多根轴线对齐方式”

    属性：

    • stretch—-“延伸” 默认值
    • flex-start —-“靠向交叉轴起点”
    • flex-end —-“靠向交叉轴终点”
    • center—-“居中”
    • space-between —-“交叉轴两端对齐”
    • space-around —-“交叉轴等间隔对齐”

  6. flex-flow —-“方向 换行” 简写

#3. flex项目（flex item）的属性：

  1. order —-“排列顺序，值小靠前”

  2. align-self —-“单个项目在交叉轴方向上的对齐方式”

  • 默认值auto，表示继承父元素的align-items属性
  1. flex-basis —-“main-size值”
  • flext-size未设定值：默认值为auto
  1. flex-grow —-“增长比例（空间过多时）”
  • 默认值为0，空间有多余时，项目也不放大；设置为1，项目可能放大
  1. flex-shrink —-“收缩比例（空间不够时）”
  • 默认值为1，空间不够时，项目自动缩小；设置为0，项目不会缩小
  1. flex —-“增长比例 收缩比例 默认值”简写
