CSS-布局之grid布局

#参考资料：MDN文档，知乎-若愚-文章

若愚：CSS Grid 系列(上)-Grid布局完整指南

#1、预备知识：

  1. 网格线：分界线组成网格，可以是水平的也可以是垂直的
  2. 网格单元：网格单元是网格中最小的单位，类似表格中的单元格
  3. 网格轨道：两个相邻网格线之间的空间，可横可竖
  4. 网格区域：网格项目向行或列方向扩展一个或多个单元，并且会创建一个网格区域
  5. 网格嵌套：一个网格项目也可以成为一个网格容器，嵌套的网格和它的父级没有关系

#2、grid容器（grid container）的属性：

  1. grid-template-areas —-“布局可视化”

  2. grid—-“简写：先行后列”

    • grid-template-rows —-“行高”
    • grid-template-columns —-“列宽”

  3. grid-gap —-“简写：先行后列”

    • grid-column-gap —-“列间距”
    • grid-row-gap —-“行间距”

  4. justify-content —-“容器里网格 行轴对齐方式”
    align-content —-“容器里网格 竖轴对齐方式”

    属性值：

    • start —-“左对齐、上对齐”
    • end —-“右对齐、下对齐”
    • center —-“居中对齐”
    • space-around —-“项目的四周空间”
    • space-between —-“项目的相邻空间”
    • space-evenly —-“项目的均匀空间”

  5. justify-items —-“项目里内容 行轴对齐方式”
    align-items —-“项目里内容 竖轴对齐方式”

    属性值：

    • start —-“”
    • end —-“”
    • center —-“”
    • stretch —-“”

  6. grid-auto-columns —-“隐式网格列宽”
    grid-auto-rows —-“隐式网格行高”

  7. grid-auto-flow —-“项目自动放置”

  8. repeat—-“重复宽度”

  1. fr—-“自由空间比例单位”
  1. minmax(min, max) —-“参数：最小值，最大值”

#3. grid项目（grid item）的属性：

  1. grid-area —-“项目区域命令”

  2. grid-column —-“简写：start+end”
    grid-row —-“简写：start+end”

    • grid-column-start —-“项目竖轴上起始线”
    • grid-column-end —-“项目竖轴上结束线”
    • grid-row-start —-“项目行轴上起始线”
    • grid-row-end —-“项目行轴上结束线”
  1. justify-self —-“单个儿项目里内容行轴对齐方式”
    align-self —-“单个儿项目里内容竖轴对齐方式”

#4. 使用习惯：

  1. 搭配方案1：
    容器的grid-template
    项目的grid-area
  1. 搭配方案2：
    容器的grid-template-columns、grid-template-rows，简写属性grid
    项目的grid-row、grid-column
